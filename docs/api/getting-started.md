# Getting Started

To explore the specific endpoints and technical details, please refer to our [API reference](/api). 

## What you will need

Our APIs require a personal access token (PAT) to authenticate your requests. You can review the [following guide](https://support.fusionsignage.com.au/en/articles/12636349-personal-access-tokens) to create your first PAT.
