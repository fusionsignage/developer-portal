# Development

As long as you [respect the capabilities of your target device](./troubleshooting.md#minimum-browser-versions-per-device), you can use any front-end development library you like,
such as React, Vue, HTMX, etc. 

If you have a static website builder that exports the website assets that works too.

Fusion Signage does not recommend one particular way of developing custom apps. It obviously helps if you have experience as a developer. 
You may also like to try using agentic tools to automatically generate a website for you.
